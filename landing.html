§ 02 what is piki

The full picture

Piki is a TypeScript monorepo that builds an interactive coding agent, its runtime, model integrations, and terminal UI. It keeps the Pi harness foundation while adding branded packages, providers, worker runtime services, and event-core projections.

The primary output is @piki/coding-agent — a CLI that provides an interactive terminal-based coding agent capable of file reads and writes, search, shell execution, web fetch, task coordination, checkpoints, and restore flows. Built on Effect, every layer is composable and testable.
